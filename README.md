# Shopee E-Commerce Sales Dashboard

## Overview

This project analyzes Shopee product sales data using Power BI.

## Dashboard Features

- Revenue Analysis
- Product Analysis
- Location Analysis
- Product Trend Analysis
- Interactive Filters

## Tools Used

- Power BI
- Power Query
- DAX

## Key Insights

- Home Appliances generated the highest revenue.
- Women's Clothing had the highest product count.
- Selangor contributed the most listings.
